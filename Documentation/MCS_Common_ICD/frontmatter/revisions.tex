    \begin{minipage}[t]{\linewidth}

    \begin{minipage}[t]{\linewidth}

            \item Fixed error where SUBSYSTEM and SERIALNO were both index 1.4
            \item Settled on UDP over TCP
            \item Maximum payload set to 4050 bytes
            \item Added DATALEN field
            \item MJD/MPM interpreted as send time
            \item Added CKSUM field
            \item Standardized response DATA field
            \item Clarified multi-packet messages
            \item Changed space notation in examples
        \end{itemize}

            \item Multipacket messages removed
            \item Fixed RPT response examples
            \item Guidance on MJD/MPM meanings
